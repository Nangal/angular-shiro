lacksRole
service in module angularShiro.directives

Description

Display if the current Subject is NOT assigned the specified role (for example, ADMIN)

Usage

as attribute
<ANY lacks-role="{string |
expression}">
   ...
</ANY>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 600.

Parameters

ParamTypeDetails
lacksRolestring expression

the excluded role (for example, ADMIN)
