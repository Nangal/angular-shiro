usernamePasswordForm
service in module angularShiro.directives

Description

The usernamePasswordForm directive will display a simple username / password form with built-in integration with angular-shiro authentication mechanism.

Style

Form markup is based on latest Bootstrap form documentation

Default labels

{
'field.username.placeholder' : 'username',
'field.password.placeholder' : 'password',
'button.submit.label' : 'Connection',
'connection.denied.message' : 'Username and password do not match'
}

Usage

as element:
<username-password-form
       [labels="{object}"]
       [on-success="{expression}"]
       [on-error="{expression}"]>
</username-password-form>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 600.

Parameters

ParamTypeDetails
labels
(optional)
object

custom labels

onSuccess
(optional)
expression

Expression to evaluate upon successful authentication

onError
(optional)
expression

Expression to evaluate upon authentication failure
