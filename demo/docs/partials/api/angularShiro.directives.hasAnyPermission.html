hasAnyPermission
service in module angularShiro.directives

Description

Display if the current Subject is assigned any of the specified permissions (for example, ['newsletter:read','newsletter:edit'])

Usage

as attribute
<ANY has-any-permission="{string |
array | expression}">
   ...
</ANY>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 600.

Parameters

ParamTypeDetails
hasAnyPermissionstring array expression

the specified specified permissions (for example, ['newsletter:read','newsletter:edit'])
