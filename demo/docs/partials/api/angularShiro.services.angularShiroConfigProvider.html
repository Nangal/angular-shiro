angularShiroConfigProvider
service in module angularShiro.services

Description

angularShiroConfigProvider is used to configure angular-shiro.

Methods

  • setAuthenticateUrl(authenticateUrl)

    The authenticateUrl specifies the url to call when the user try to login to the application. The url is called with a POST method.

    Parameters
    ParamTypeDetails
    authenticateUrlstring

    The uri to be called on a login attempt (default '/api/authenticate')

    Example

    app.config([ 'angularShiroConfigProvider', function(config) {
    config.setAuthenticateUrl('/my/custom/url');
} ]);

  • setFilter(path, filterName)

    Associate the specified filter(s) with the given path

    Parameters
    ParamTypeDetails
    pathstring

    the path for which the filter(s) should be applied

    filterNamestringarray

    the filter(s) to apply to the path

    Example

    app.config([ 'angularShiroConfigProvider', function(config) {
    config.setFilter('/my/path', 'authc');
} ]);

  • setLoginPath(loginPath)

    The loginPath specifies the path used for redirecting the user on an attempt to reach a denied url.

    Parameters
    ParamTypeDetails
    loginPathstring

    The path to the login page (default '/login')

    Example

    app.config([ 'angularShiroConfigProvider', function(config) {
    config.setLoginPath('/my/login/page');
} ]);

  • setLogoutPath(logoutPath)

    The logoutPath specifies the path to apply to $location when the user log out of the application.

    Parameters
    ParamTypeDetails
    logoutPathstring

    the application path to be applied on log out

    Example

    app.config([ 'angularShiroConfigProvider', function(config) {
    config.setLogoutPath('/my/logout/page');
} ]);
