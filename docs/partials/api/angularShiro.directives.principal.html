principal
service in module angularShiro.directives

Description

The principal directive will output the Subject's principal or a property of that principal.

If no property attribute is specified the directive will render the angular.toJson(principal) value.

Usage

as element:
<principal
       [property="{string}"]>
</principal>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 600.

Parameters

ParamTypeDetails
property
(optional)
string

a property name of the principal object
