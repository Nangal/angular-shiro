AuthorizationInfo
service in module angularShiro.services

Description

AuthorizationInfo represents a Subject's authorization informations (roles, permissions, etc) used for access control operations

Usage

AuthorizationInfo(roles, permissions);

Parameters

ParamTypeDetails

the list of the Subject roles
