hasRole
service in module angularShiro.directives

Description

Display if the current Subject has the specified role (for example, ADMIN)

Usage

as attribute
<ANY has-role="{string | expression}">
   ...
</ANY>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 600.

Parameters

ParamTypeDetails
hasRolestring expression

the role to check (for example, ADMIN)
